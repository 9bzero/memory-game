# memory-game

Flip cards, find matching pairs.

## Difficulty levels

| Level | Grid | Pairs |
|---|---|---|
| Easy | 4×3 | 6 |
| Medium | 4×4 | 8 |
| Hard | 6×4 | 12 |

Cards flip back after 1 second if no match. Matched cards stay face-up. Timer and move counter track your run.

## Run

```bash
npm install && npm run dev
```
